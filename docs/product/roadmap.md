# Product Roadmap

## Completed MVP Foundations

- Documentation information architecture.
- Tauri, React, TypeScript, and Vite scaffold.
- Local repository selection.
- SQLite-backed persistence for repositories, indexing jobs, approval requests, and indexed file facts.
- Git metadata loading for selected repositories.
- File-tree indexing with ignore rules.
- Indexed-file browsing with extension filters and selected-file detail.
- Safe file content preview with size, binary, outside-repository, loading, and unavailable states.
- Warm premium dashboard visual system for the desktop shell.

## Next Product Priorities

- Add file open/reveal actions that preserve Tauri safety boundaries.
- Add repository rescan controls and freshness metadata.
- Add package/script detection to the indexing pipeline.
- Add provider-backed planning runs behind human approval.
- Expand approval queue detail views with diffs, risks, and validation evidence.
